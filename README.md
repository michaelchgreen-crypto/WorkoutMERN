# WorkoutMERN

This is a MERN app, the file structiure is split between the front end and the back end.
The node modules files have been omitted to save space.

To run, you will need to use your own MongoDB Atlas Cluster.
  Navigate to the .env file in the Backend folder and replace.

Start a terninal in the Backend and run "npm install" and "npm start"
  If you have properly entered your cluster information you should see a success message.
Start another terminal at frontend/gym and run the command "npm run dev"
  The application will run in the browser at http://localhost:5173/
